# Fisiostep – Landing Page

Landing page per **Fisiostep - Dott. Elia Manuel Osteopata**, studio di osteopatia e massoterapia a Montalto Uffugo (CS).

## Anteprima locale

```bash
python3 -m http.server 8080
```

Visita [http://localhost:8080](http://localhost:8080).

## Deploy

Il sito è configurato per **GitHub Pages**. Dopo il push su `main`, la pagina sarà disponibile all'URL del repository.

## Struttura

- `index.html` – pagina principale
- `styles.css` – stili e animazioni
- `script.js` – navigazione, scroll reveal, menu mobile
